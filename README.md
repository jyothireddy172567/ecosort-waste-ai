🌱 EcoSort AI

🚀 Project Overview

EcoSort AI is an AI-based smart waste segregation web application that helps users identify and classify waste materials using image input. The system promotes environmental awareness by guiding users on proper waste disposal.

---

🎯 Objectives

- Automatically classify waste using AI
- Help users understand waste categories
- Provide nearby waste disposal/recycling centers
- Promote eco-friendly habits

---

🧠 Features

- 📷 Upload or capture waste images
- 🤖 AI-based waste classification
- 🗂️ Categorization (Plastic, Organic, Metal, etc.)
- 📍 Nearby waste centers (India-based default)
- 🌐 Deployed web application
- 🎨 Custom UI with EcoSort branding

---

🛠️ Technologies Used

- Frontend: React.js + Vite
- Styling: Tailwind CSS
- AI Integration: Lovable AI (Vision-based classification)
- Version Control: GitHub
- Deployment: Vercel

---

📂 Project Structure

- "public/" → Static assets (images, favicon)
- "src/" → Main application code (components, pages, logic)
- "components/" → Reusable UI components
- "index.html" → Root HTML file
- "package.json" → Project dependencies and scripts

---

🧪 Dataset & Model

- Model Used: Pre-built Vision AI model (Lovable AI)
- Dataset: General waste image dataset (plastic, organic, metal, etc.)
- Training: Not trained manually (uses pre-trained AI model)

---

⚙️ How It Works

1. User uploads an image
2. Image is sent to AI model
3. AI analyzes and classifies waste
4. Result is displayed with category
5. Nearby disposal centers are suggested

---

🌍 Deployment

- Live URL: https://ecosort-waste-ai.lovable.app
